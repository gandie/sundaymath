# Binomische Formeln

1. (a+b)²     = a² + 2ab + b²
2. (a-b)²     = a² - 2ab + b²
3. (a+b)(a-b) = a² - b²

## Für'n Spaß

1. (klim+bim)²          = klim² + 2klimbim + bim²
2. (klim-bim)²          = klim² - 2klimbim + bim²
3. (klim+bim)(klim-bim) = klim² - bim²
